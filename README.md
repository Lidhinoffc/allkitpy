# Allpy

Allpy is an all-in-one Python utility library.

## Features

- 🖼 Image Processing
- 📄 PDF Tools
- 🎵 Audio Utilities
- 🎬 Video Utilities
- 🌐 Web Utilities
- 📊 Data Processing
- 💻 System Utilities
- 🤖 AI Providers

## Installation

```bash
pip install allpy
```

## Example

```python
import allpy

print(allpy.text.uppercase("hello"))
```

Output:

```
HELLO
```

## Python Version

Python 3.10+
